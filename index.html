¿Para que sirve la etiqueta H1?

La etiqueta H1 sirve para indicar el titulo principal de la pagina web

¿Para que sirve la etiqueta P?

La etiqueta P sirve para indicar un parrafo de texto

¿Para que sirve la etiqueta H2?

La etiqueta H2 sirve para indicar un subtitulo de la pagina web

Tambien existen las listas ordenadas y desordenadas

¿Para que sirve la etiqueta UL?

La etiqueta UL sirve para indicar una lista desordenada

  • Arroz
  • Frijoles
  • Lentejas

¿Para que sirve la etiqueta OL?

La etiqueta OL sirve para indicar una lista ordenada

  1. Arroz
  2. Frijoles
  3. Lentejas

¿Para que sirve la etiqueta IMG?

La etiqueta IMG sirve para indicar una imagen

¿Para que sirve la etiqueta A?

La etiqueta A sirve para indicar un enlace
